# Finance Manager v4

Персональный финансовый менеджер с красивым интерфейсом.

## Функции

- ✅ Доход/расход
- ✅ 5 категорий (АЗС, Расходники, Аренда, Продукты, Другое)
- ✅ Добавление новых категорий
- ✅ История операций с поиском
- ✅ Несколько кошельков
- ✅ Аналитика по датам
- ✅ Фото чеков
- ✅ Экспорт/импорт JSON

## Технологии

- React 18
- Tailwind CSS
- localStorage для сохранения данных

## Использование

Откройте `index.html` в браузере или разверните на Vercel.

## Развертывание на Vercel

1. Загрузите репозиторий на GitHub
1. Подключите репозиторий к Vercel
1. Vercel автоматически развернет приложение
